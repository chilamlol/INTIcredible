# INTIcredibles

This is the backend API for all the service in INTIcredible app and admin web portal. The API are built using Python with Flask frame work.

**Prerequisite**

* Python 3.7
* Flask
  * Flask-MySQL
  * Flask-Mail
* PyMySQL
* Hashlib
* MySQL 5.7 and above
* PyJWT

API key is using JWT hashed GUID.
